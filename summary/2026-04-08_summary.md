## 开发内容总览

1. **文件上传功能**：支持 PDF、DOCX、TXT 格式文件上传至 `data/` 目录并自动加入向量数据库

## 详细代码变更

### 🎯 核心功能模块

#### 文件上传与向量库集成
- **文件**: `rag/vector_store.py`
- **变更类型**: 新增
- **变更内容**:
  在 `VectorStoreService` 类中新增 `add_document` 方法，支持将文本内容直接添加到向量数据库：
  ```python
  def add_document(self, text: str, filename: str):
      """
      将文本内容添加到向量库
      :param text: 文本内容
      :param filename: 文件名（用于metadata）
      :return: 是否成功
      """
      from langchain_core.documents import Document
      doc = Document(page_content=text, metadata={"source": filename})
      split_docs = self.spliter.split_documents([doc])
      if not split_docs:
          logger.warning(f"[添加文档]{filename}分片后没有有效内容")
          return False
      self.vector_store.add_documents(split_docs)
      logger.info(f"[添加文档]{filename}添加成功")
      return True
  ```
- **影响范围**: 整个 RAG 系统，现已支持运行时动态添加文档

#### Streamlit 文件上传界面
- **文件**: `app.py`
- **变更类型**: 新增
- **变更内容**:
  1. 新增 `extract_text_from_file()` 函数，支持从 PDF、DOCX、TXT 文件提取文本
  2. 新增 `show_file_uploader()` 函数，在侧边栏提供两个上传入口：
     - **文件上传器**：支持选择本地文件（PDF/DOCX/TXT）
     - **文本粘贴上传**：文本输入框 + 文件名输入框 + 提交按钮
  3. 上传流程：提取文本 → 保存到 `data/` 目录 → 调用 `VectorStoreService.add_document()` 添加到向量库
- **影响范围**: Streamlit 前端界面

## 技术亮点
- **RAG 动态扩展**：无需重启服务即可向知识库添加新内容
- **多格式支持**：统一处理 PDF、DOCX、TXT 三种文档格式

- **向量数据库集成**：使用 Chroma 的 `RecursiveCharacterTextSplitter` 进行智能分块
